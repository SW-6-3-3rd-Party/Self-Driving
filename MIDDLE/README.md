# MIDDLE

RPi #2 중간 계층 코드 묶음입니다. 실제 실행 파일과 시험 코드는
[`RPI2/`](./RPI2/)에 두고, 이 폴더는 그 상위 진입점 역할만 합니다.

포함 범위:

- `RPI2/app.py`
- `RPI2/protocol.py`
- `RPI2/lane_detector.py`
- `RPI2/ultrasonic.py`
- `RPI2/udp_receiver.py`
- `RPI2/run_rpi2.sh`
- `RPI2/stop_rpi2.sh`
- `RPI2/tests/`
- `RPI2/README.md`

배포나 검증 시에는 `RPI2/README.md`의 절차를 따른다.
