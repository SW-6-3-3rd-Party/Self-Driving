# Legacy AEB Demo

This file is kept only as an old standalone AEB demo reference.

The current modular runtime is:

```bash
python3 -m HPVC.hpvc_aeb
```

Use `HPVC/hpvc_aeb.py` for the active HPVC AEB path. The legacy demo may still
mention direct Raspberry Pi sensors and old bench assumptions, so do not treat it
as the current module contract.
