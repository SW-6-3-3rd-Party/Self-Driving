# HPVC Legacy References

These files are preserved from earlier standalone AEB/LKAS experiments.

They are not the active module contracts. The current paths are:

- HPVC AEB runtime: `HPVC/hpvc_aeb.py`
- MIDDLE perception packet: `MIDDLE/protocol.py`
- HPVC steering packet encoder: `HPVC/frontEncodeSteeringPacket.m`

Keep new HPVC control work in the HPVC module root or in a named HPVC
subdirectory, not in these legacy references.
